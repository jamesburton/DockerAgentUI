1. Add a `HashSet<SessionEvent>` field to track expanded events:
```csharp
private readonly HashSet<SessionEvent> _expandedEvents = [];
```

2. Add a toggle method:
```csharp
private void ToggleMeta(SessionEvent evt)
{
    if (!_expandedEvents.Remove(evt))
        _expandedEvents.Add(evt);
}
```

3. Update the event rendering loop. Wrap each event in a clickable div and conditionally show meta panel:
```razor
@foreach (var evt in Events)
{
    <div class="terminal-event-row" @onclick="() => ToggleMeta(evt)"
         style="cursor: @(evt.Meta is { Count: > 0 } ? "pointer" : "default")">
        <pre class="@GetEventClass(evt.Kind)">@evt.Data</pre>
    </div>
    @if (_expandedEvents.Contains(evt) && evt.Meta is { Count: > 0 })
    {
        <div class="terminal-meta-panel" style="padding: 4px 16px 8px 24px; background: rgba(255,255,255,0.03); border-left: 2px solid #555; margin-bottom: 4px; font-size: 0.85em;">
            @foreach (var kv in evt.Meta)
            {
                <span style="color: #888; margin-right: 4px;">@kv.Key:</span>
                <span style="color: #aaa; margin-right: 12px;">@kv.Value</span>
            }
        </div>
    }
}
```

4. Keep the anchor div for auto-scroll after the foreach loop (unchanged).

Note: Using inline styles consistent with the existing terminal CSS approach in this component. The dark background, muted colors, and left border give a visual indent effect matching the terminal aesthetic.
