  <name>Task 2: Session detail page with live streaming and history replay</name>
  <files>
    src/AgentHub.Web/Components/Pages/SessionDetail.razor
  </files>
  <action>
    Create `src/AgentHub.Web/Components/Pages/SessionDetail.razor`:
    Route: @page "/session/{SessionId}"

    Inject: DashboardApiClient (as Api), SseStreamService (as SseStream), NavigationManager, ISnackbar

    Parameters:
    - `[Parameter] public string SessionId { get; set; }`

    State:
    - `SessionSummary? _session`
    - `List<SessionEvent> _events = new()`
    - `CancellationTokenSource _cts = new()`
    - `bool _loading = true`
    - `List<SessionEvent> _pendingApprovals = new()` (unresolved ApprovalRequest events)

    OnInitializedAsync:
    1. Load session metadata: `_session = await Api.GetSessionAsync(SessionId)`
    2. If session is null, show error and return
    3. If session state is Running or Pending:
       - Start SSE stream: `var reader = SseStream.SubscribeSession(SessionId, _cts.Token)`
       - Fire-and-forget background task to read from reader:
         ```csharp
         _ = Task.Run(async () =>
         {
             await foreach (var evt in reader.ReadAllAsync(_cts.Token))
             {
                 _events.Add(evt);
                 if (evt.Kind == SessionEventKind.ApprovalRequest)
                     _pendingApprovals.Add(evt);
                 if (evt.Kind == SessionEventKind.StateChanged)
                 {
                     // Reload session metadata to get updated state
                     _session = await Api.GetSessionAsync(SessionId);
                 }
                 await InvokeAsync(StateHasChanged);
             }
         }, _cts.Token);
         ```
    4. If session state is Stopped or Failed:
       - Load history with pagination: load pages of 100 events until exhausted
       ```csharp
       int page = 1;
       List<SessionEvent> batch;
       do
       {
           batch = await Api.GetSessionHistoryAsync(SessionId, page, 100, ct: _cts.Token);
           _events.AddRange(batch);
           page++;
       } while (batch.Count == 100);
       ```
    5. Set _loading = false, StateHasChanged

    Render:
    - If _loading: MudProgressCircular
    - If _session is null: MudAlert "Session not found"
    - Otherwise:
      - Metadata header (MudPaper with MudGrid):
        - MudItem: Status as MudChip with color (same mapping as SessionTable)
        - MudItem: Agent Type (Backend)
        - MudItem: Host (Node ?? "Unassigned")
        - MudItem: Duration (computed from CreatedUtc)
        - MudItem: Prompt (full text, no truncation)
        - MudItem: Started (CreatedUtc formatted)
      - Action buttons:
        - MudButton "Back to Fleet" -> NavigationManager.NavigateTo("/")
        - If Running/Pending: MudButton "Stop Session" (Color.Error) -> StopSession
        - If Running/Pending: MudButton "Force Stop" (Color.Error, Variant.Outlined) -> ForceStopSession
      - If _pendingApprovals.Any(): render ApprovalAlert for each pending approval
      - TerminalOutput component: Events=_events, SessionId=SessionId

    StopSession method:
    - Call Api.StopSessionAsync(SessionId, force: false)
    - Snackbar success message
    - Reload session metadata

    ForceStopSession method:
    - Call Api.StopSessionAsync(SessionId, force: true)
    - Snackbar success message
    - Reload session metadata

    HandleApprovalResolved(SessionEvent evt, bool approved):
    - Extract approvalId from evt.Meta["approvalId"] (if present, else use evt.Data as fallback)
    - Call Api.ResolveApprovalAsync(approvalId, approved)
    - Remove from _pendingApprovals
    - Snackbar success: "Approval {approved ? "granted" : "rejected"}"

    Disposal:
    - @implements IDisposable
    - In Dispose: cancel _cts to stop SSE stream
  </action>
