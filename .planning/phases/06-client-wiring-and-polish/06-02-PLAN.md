5. **Create HostMetricPollingService** in `src/AgentHub.Orchestration/Monitoring/HostMetricPollingService.cs`:
   - Follows SessionMonitorService pattern: `BackgroundService` with `IDbContextFactory`, `ISshHostConnectionFactory`, `DurableEventService`
   - Constructor params: `IDbContextFactory<AgentHubDbContext>`, `ISshHostConnectionFactory`, `DurableEventService`, `ILogger<HostMetricPollingService>`
   - `_pollInterval = TimeSpan.FromSeconds(30)`
   - `ExecuteAsync`: loop with `await PollAllHostsAsync(ct)` then `await Task.Delay(_pollInterval, ct)`
   - `PollAllHostsAsync`: query all enabled hosts from DB, `PollHostAsync` each (with try/catch per host so one failure doesn't block others)
   - `PollHostAsync(HostEntity host, CancellationToken ct)`:
     a. Create SSH connection via factory: `var conn = _connectionFactory.Create(host.Address!, host.HostId);`
     b. Detect OS from `host.Os` field (case-insensitive contains: "windows", "linux", "macos"/"darwin")
     c. Run OS-appropriate SSH command:
        - **Windows**: `powershell -Command "$cpu=(Get-Counter '\\Processor(_Total)\\% Processor Time').CounterSamples[0].CookedValue; $os=Get-CimInstance Win32_OperatingSystem; \"$cpu|$([math]::Round($os.TotalVisibleMemorySize/1024))|$([math]::Round(($os.TotalVisibleMemorySize-$os.FreePhysicalMemory)/1024))\""`
        - **Linux**: `bash -c "cpu=$(grep 'cpu ' /proc/stat | awk '{u=$2+$4; t=$2+$4+$5} END {printf \"%.1f\", u*100/t}'); mem=$(free -m | awk '/Mem:/ {print $2\"|\"$3}'); echo \"$cpu|$mem\""`
        - **macOS**: `bash -c "cpu=$(top -l 1 -n 0 | grep 'CPU usage' | awk '{print $3}' | tr -d '%'); mem_total=$(sysctl -n hw.memsize | awk '{print int($1/1048576)}'); mem_used=$(vm_stat | awk '/Pages active|Pages wired/ {sum+=$NF} END {print int(sum*4096/1048576)}'); echo \"$cpu|$mem_total|$mem_used\""`
     d. Parse pipe-delimited output: `cpu|memTotal|memUsed` (order varies by OS, normalize)
     e. Update HostEntity: set CpuPercent, MemUsedMb, MemTotalMb, MetricsUpdatedUtc = DateTimeOffset.UtcNow
     f. SaveChangesAsync
     g. Emit HostMetrics SSE event via DurableEventService.PublishAsync:
        ```csharp
        await _events.PublishAsync(new SessionEvent(
            SessionId: "",  // Host-level event, no session
            Kind: SessionEventKind.HostMetrics,
            TsUtc: DateTimeOffset.UtcNow,
            Data: $"Host {host.HostId} metrics updated",
            Meta: new Dictionary<string, string>
            {
                ["hostId"] = host.HostId,
                ["cpu"] = entity.CpuPercent?.ToString("F1") ?? "",
                ["memUsedMb"] = entity.MemUsedMb?.ToString() ?? "",
                ["memTotalMb"] = entity.MemTotalMb?.ToString() ?? ""
            }), ct);
        ```
   - Expose `public async Task RunOnceAsync(CancellationToken ct)` for testability (same as SessionMonitorService pattern)
