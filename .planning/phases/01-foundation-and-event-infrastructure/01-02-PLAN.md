<objective>
Replace the in-memory SessionEventBus with a durable event service that persists events to the database and supports SSE streaming with Last-Event-ID replay. Add fleet-wide SSE endpoint.

Purpose: Real-time event streaming with replay is the backbone of the monitoring system. Without durable events, clients that disconnect lose all events that occurred while offline. The fleet-wide endpoint enables the Phase 4 dashboard.
Output: DurableEventService replacing SessionEventBus, two SSE endpoints (per-session + fleet-wide) with replay support, and comprehensive tests.
</objective>

       - `SubscribeFleet(string? lastEventId, CancellationToken ct) -> IAsyncEnumerable<SseItem<SessionEvent>>`:
         a. Same replay pattern but without SessionId filter
         b. Same channel/subscribe/cleanup pattern
