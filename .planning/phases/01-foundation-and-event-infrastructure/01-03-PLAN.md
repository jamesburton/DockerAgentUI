<objective>
Create the agent adapter abstraction with Claude Code as the first implementation, define the host daemon command protocol, and set up per-agent configuration.

Purpose: The adapter pattern decouples "what agent to run" from "where to run it", making it trivial to add new agent types in Phase 2+. The host daemon protocol defines how the coordinator will communicate with remote hosts (Phase 2 implements execution over SSH).
Output: IAgentAdapter interface, ClaudeCodeAdapter using CliWrap, AgentAdapterRegistry, host daemon JSON-over-SSH protocol definition, agent config files, and tests.
</objective>
